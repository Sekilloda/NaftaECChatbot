- install homebrew
- install ngrok "brew install ngrok"
- run authtoken command for ngrok as per https://dashboard.ngrok.com/get-started/setup/macos
- install tesseract "brew install tesseract"
- install tesseract language packs with "brew install tesseract-lang"
- install pyenv "brew install pyenv"
- install virtual environments for python "brew install pyenv-virtualenv"
- install python version 3.11 "pyenv install 3.11"
- clone github repository and cd into repository folder "Local"
- create virtual environment "pyenv virtualenv 3.11 nafta"
- activate virtual environment "pyenv local nafta" and make sure (nafta) is prepended to prompt
- install requirements with "pip install -r requirements_macos.txt "
- run ngrok in another terminal window with "ngrok http --url=YOURURL 5001"
- run gunicorn in the same terminal with the (nafta) prompt "gunicorn -w 4 -b 0.0.0.0:5001 app_mistral:app"
