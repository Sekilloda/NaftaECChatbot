- Make an account in ngrok
- Download ngrok and add authtoken as per https://dashboard.ngrok.com/get-started/setup/windows
- Run ngrok, then in the CMD window that opens run "ngrok http --url=discrete-physically-colt.ngrok-free.app 5001"
- Install python 3.13.5 
- Install tesseract https://github.com/UB-Mannheim/tesseract/wiki with addiditonal language data and additional script data.
- Add tesseract installation directory to PATH.
- Install git and clone the repository https://github.com/Sekilloda/NaftaECChatbot.git
- Create .env file and populate with API keys.
- Open a command line on the repository directory
- Run "Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser"
- Run "python -m venv venv"
- Run "cd ./venv/Scripts/activate" and check prompt has "(venv)" at the start of it.
- Run "pip install -r requirements_windows.txt
- Run "waitress-serve --host=0.0.0.0 --port=5001 app_mistral:app"
